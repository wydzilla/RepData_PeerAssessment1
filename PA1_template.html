What is mean total number of steps taken per day?

To calculate this value, first, we need calculate number of steps per each day. Function aggregate breaks data into groups. We will group steps variable into groups with the same date (date column). For each group we will apply sum function to calculate total steps taken each day.

steps_per_day <- aggregate(steps ~ date, activity_data, sum)

Strategy for filling missing values

If there is no value of steps for given row, we are going to use mean for this interval averaged over all the days. Those were calculated for Daily activity pattern.

for (i in 1:length(all_activity_data$steps)) {
    if (is.na(all_activity_data$steps[i])) { 
        this_interval <- all_activity_data$interval[i]
        all_activity_data$steps[i] <- avg_daily_activity$steps[avg_daily_activity$interval == this_interval] #use interval mean 
    }

}
